## API

### UxDivider

#### Props

| 参数名      | 描述                                         | 类型    | 默认         |
| ----------- | -------------------------------------------- | ------- | ------------ |
| type        | 分割线类型，可选值：`horizontal`、`vertical` | String  | `horizontal` |
| dashed      | 是否是虚线                                   | Boolean | false        |
| orientation | 分割线文字位置，可选值:`left`、`right`       | String  | ''           |
