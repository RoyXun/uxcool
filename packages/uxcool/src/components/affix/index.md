## API

### UxAffix

#### Props

| 参数名        | 描述                                   | 类型   | 默认 |
| ------------- | -------------------------------------- | ------ | ---- |
| offset-top    | 滚动距离大于元素上部指定距离时触发浮动 | Number |      |
| offset-bottom | 滚动距离小于元素底部指定距离时触发浮动 | Number |      |

> 只有设置了`offset-top`或`offset-bottom`时,`affix`才会起作用

#### Events

| 事件名 | 描述               | 回调              |
| ------ | ------------------ | ----------------- |
| change | 浮动状态切换时触发 | Function(affixed) |
