## API

### UxAvatar

#### Props

| 参数名 | 描述                                   | 类型   | 默认    |
| ------ | -------------------------------------- | ------ | ------- |
| size   | 大小,可选值: `default`,`large`,`small` | String | default |
| shape  | 形状,可选值: `circle`,`square`         | String | circle  |
| src    | 头像图片 url                           | String |         |
| icon   | 头像 icon                              | String |         |
